This folder contains processed data for demonstration purpose.
- The <cchs2015_demonstration> data has been created based on the HS data set (/CCHS_Nutrition_2015_PUMF/Data_Donnee/hs.sas7bdat). The steps leading to the dataset in the present folder are described in <0.1_Prepare_CCHS_data.R>.
